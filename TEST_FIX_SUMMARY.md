# Playwright Test Fix Summary

## Problem Identified
Tests were timing out because they used incorrect selectors that didn't match the actual HTML generated by UIController.js.

## Root Cause
The test file was using guessed selectors instead of reading the actual source code to find the correct ones.

## Solution Implemented

### Step 1: Read Actual HTML Structure
Examined these files to find exact selectors:
- `index.html` - Landing screen structure
- `js/engine/UIController.js` - Dynamic HTML generation
- `js/content/ui-content.js` - Button text content

### Step 2: Documented Exact Selectors
1. **Start button**: `#begin-button` (text: "Begin Experience")
2. **Role selection container**: `#role-cards-container`
3. **Role buttons**: `.role-select-button[data-role-id="american-sailor"]` etc.
4. **Narrative text**: `#scene-narrative p`
5. **Choice buttons**: `.choice-button`

### Step 3: Rewrote Navigation Helpers
```javascript
async function navigateToRoleSelect(page) {
  await setupPage(page);
  await page.waitForSelector('#begin-button', { state: 'visible', timeout: 15000 });
  await page.click('#begin-button');
  // ... navigate to role selection
  await page.waitForSelector('#role-cards-container', { state: 'visible', timeout: 15000 });
}

async function navigateToRole(page, role) {
  await navigateToRoleSelect(page);
  const selectors = {
    sailor: '.role-select-button[data-role-id="american-sailor"]',
    aviator: '.role-select-button[data-role-id="japanese-aviator"]',
    civilian: '.role-select-button[data-role-id="american-civilian"]'
  };
  await page.waitForSelector(selectors[role], { state: 'visible', timeout: 10000 });
  await page.click(selectors[role]);
  await page.waitForSelector('#scene-narrative p', { state: 'visible', timeout: 15000 });
}
```

### Step 4: Fixed Playwright Configuration
- Changed HTML reporter output to `playwright-report` to avoid conflicts
- Fixed webServer config: `command: 'npx serve . -l 8000', port: 8000`
- Set global timeout to 60000ms
- Updated `.gitignore` to exclude test artifacts

## Remaining Issue
The choice buttons are disabled while the typewriter effect is running. The test needs to wait for buttons to be enabled before clicking.

### Fix Needed
```javascript
// Wait for choice to be enabled, not just visible
await page.locator('.choice-button').first().waitFor({ state: 'visible', timeout: 5000 });
await page.waitForFunction(() => {
  const btn = document.querySelector('.choice-button');
  return btn && !btn.disabled;
}, { timeout: 10000 });
await page.locator('.choice-button').first().click();
```
