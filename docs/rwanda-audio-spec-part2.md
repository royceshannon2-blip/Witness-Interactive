**Sound 2: Urban Morning Panic**

### Scene: rw-ts-scene-02a
**Context**: Church. Hundreds sheltering. Militia attacks.

**Sound 1: Church Interior Ambient (Before Attack)**
- Type: Ambient loop
- Description: Crowd murmuring, children crying, prayers (Kinyarwanda), footsteps echoing, tense breathing
- Mood: Fearful, crowded, tense
- Priority: HIGH
- Notes: RWANDA-SPECIFIC: Catholic church acoustics. Hundreds of people. Should feel claustrophobic, scared.
- Pearl Harbor overlap: None

**Sound 2: Distant Gunshot**
- Type: One-shot
- Description: Single gunshot, muffled by walls, outside church
- Mood: Shocking, violent
- Priority: HIGH
- Notes: Marks Father Michel's death. Should be sudden, shocking. Not graphic but impactful.
- Pearl Harbor overlap: Could adapt gunshot sound if available


**Sound 3: Grenade Explosion (Distant/Muffled)**
- Type: One-shot
- Description: Grenade explosion inside church, muffled if player is in crawlspace
- Mood: Terrifying, violent
- Priority: HIGH
- Notes: TRAUMA-INFORMED: Should be impactful but not graphic. If player hides in ceiling, sound is muffled.
- Pearl Harbor overlap: Could adapt explosion sound from Pearl Harbor

**Sound 4: Crowd Panic (Distant)**
- Type: Ambient loop (brief)
- Description: Screaming, running, chaos - but distant/muffled if player escapes or hides
- Mood: Terrifying
- Priority: HIGH
- Notes: TRAUMA-INFORMED: Panic sounds should be distant, brief. Never prolonged graphic violence.
- Pearl Harbor overlap: Could adapt panic sounds

### Scene: rw-ts-scene-02b

- Priority: HIGH
- Notes: SILENCE IS POWERFUL: This should be mostly quiet. Occasional sounds create tension. Less is more.
- Pearl Harbor overlap: None

**Sound 2: Roadblock Sounds (Distant)**

- Mood: Threatening, distant
- Priority: MEDIUM
- Notes: Should sound like it's 1-2 blocks away, muffled by walls. Creates constant low-level threat.
