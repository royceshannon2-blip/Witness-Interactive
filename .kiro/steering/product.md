# Witness Interactive — Product Context

Witness Interactive is a browser-based educational history game targeting 
high school and college students struggling with AP History exams. 

The MVP is Pearl Harbor (December 7, 1941) with 3 playable roles.
Long-term vision: full 3D immersive game with 50+ missions.

Core educational value: AP curriculum alignment, not just entertainment.
Every scene must connect to AP reasoning skills: causation, continuity, 
perspective, argumentation.

Target users: AP History students, college freshmen, self-directed learners.
Deployment: GitHub Pages, zero dependencies, no build step.---
