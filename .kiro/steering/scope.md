# MVP Scope Boundaries

IN SCOPE for MVP:
- Pearl Harbor mission, 3 roles, 4-5 scenes each
- Choice/consequence system
- Historical Ripple timeline
- AP Knowledge Checkpoint
- Historical Timeline mission selector (visual only)
- GitHub Pages deployment

OUT OF SCOPE for MVP:
- Backend, database, user accounts
- Real audio files
- 3D graphics or WebGPU
- Multiple simultaneous missions
- Teacher dashboard (architecture hook only)
- localStorage persistence---
inclusion: always
